Comparable_Comparator
	객체를 비교하는데 사용되는 두 가지 다른 인터페이스
	서로 다른 방식으로 객체를 비교하고 정렬하는 데 사용됨
	
Comparable
	객체 자체가 비교 가능한지 나타내는 인터페이스
	객체가 자신과 다른 객체를 비교할 때 compareTo() 메서드를 사용해서 정의
	ex) 숫자, 문자열 또는 날짜와 같은 객체를 정렬할 때 정렬 순서를 지정할 수 있음
	
Comparator
	객체들을 서로 다른 방식으로 비교하고 정렬하는데 사용
	객체 클래스에 직접 Comparable 구현하지 않고 다른 비교 규칙을 적용하고 싶을 때 사용
