데이터를 저장하기 위해 이름을 할당받은 메모리 공간을 의미하며, 이렇게 저장된 값은 변경 될 수 있다.

ex)
int a = 1;
a = 2;

변수 생성 규칙
1. 변수의 이름은 영문자(대소문자), 언더스코어(_), 달러($)로만 구성할 수 있다.
2. 변수의 이름은 숫자로 시작할 수 없다.
3. 변수 이름 사이에는 공백을 포함할 수 없다. 
4. 변수 이름으로 자바가 미리 설정한 이름은 사용할 수 없다.

변수 종류
자바의 변수는 타입에 따라서 크게 구분할 수 있다.

1. 정수형: byte(1) short(2) int(4) long(8)
2. 실수형: float(4) double(8)
3. 문자형: char(2) string()
4. 논리형: boolean(1)
