List와 ArrayList의 차이
		List = interface, 										ArrayList = Class
		인스턴스화 불가능											인스턴스화 가능
		크기가 가변적												크기가 가변적			// 가변적: 바꿀 수 있거나 바뀔 수 있다.
		요소를 추가하거나 제거할 때 크기가 동적으로 조정됨					내부적으로 배열을 사용하여 요소를 저장
																배열을 기반으로 하기에 요소에 빠르게 접근 가능
																요소를 검색하거나 인덱스로 접근할 때 성능이 우수
																삽입, 삭제 등은 요소의 이동을 동반하기 때문에 다소 느릴 수 있음
