형변환
다른 데이터 타입 간에 값을 변환하는 것을 말한다.

자동형변환(묵시적 변환)
	작성하는 방법
		변형하고자 하는 형변환 새로운 변수명 = 변경하는 형변환;
			int intNum = 100;
			Long longNum = intNum;	
	크기가 작은 것을 큰 것을 바꾸는 것
	(byte -> short)
	(short -> int)
	(int -> long)

강제형변환
	작성하고자 하는 방법
		Long longNum = 200;
		ing intNum = (int) longNum;
		크기가 큰 것을 작은 것으로 바꾸는 것
		(long -> int)
		(int -> short)
	(short -> byte)
	
문자에서 숫자로 형변환
	문자를 숫자로 형변환 할 때는 영어의 경우 고유 가지고 있는 숫자로 변경이 된다.
	문자 인코딩: 사용자가 입력한 문자나 기호들을 컴퓨터가 이용할 수 있는 신호로 만드는 것.
	ex) char charExam = 'Z';
		int intExam = charExam;
		인트의 출력값은 90이 된다. -> ASCII CODE 90 = Z이다.
